# 감염병 조사 웹사이트 📚

초등학교 3학년 학생들을 위한 감염병 조사 활동 웹사이트입니다.

## 📖 포함된 감염병

1. **코로나19** - 최근 전 세계를 힘들게 한 감염병
2. **독감** - 매년 겨울마다 유행하는 감염병
3. **콜레라** - 더러운 물로 퍼지는 무서운 병
4. **페스트(흑사병)** - 중세 시대의 무서운 감염병
5. **천연두** - 인류가 완전히 없앤 감염병
6. **결핵** - 폐를 아프게 하는 감염병

## 🌟 웹사이트 특징

- ✅ 초등학교 3학년 수준에 맞는 쉬운 설명
- ✅ 큰 글씨와 밝은 색상으로 읽기 편한 디자인
- ✅ 각 감염병별로 별도 페이지 구성
- ✅ 풍부한 사진 자료 (무료 이미지 사용)
- ✅ 링크를 통한 자유로운 탐색
- ✅ 모바일 기기에서도 잘 보임

## 📋 각 페이지 구성

각 감염병 페이지에는 다음 내용이 포함되어 있습니다:

- 감염병이 무엇인지
- 언제 유행했는지
- 어떤 피해를 주었는지
- 어떻게 예방하고 치료했는지
- 재미있는 사실

## 🚀 사용 방법

### 1. 웹사이트 열기

`index.html` 파일을 더블클릭하면 웹 브라우저에서 웹사이트가 열립니다.

### 2. 학생들에게 안내하기

학생들에게 다음과 같이 안내해주세요:

> "메인 페이지에서 궁금한 감염병을 클릭하세요.
> 각 페이지를 읽고 중요한 내용을 노트에 적어보세요.
> '다음' 버튼이나 '목록으로 돌아가기' 버튼을 눌러서 다른 감염병도 조사해보세요!"

### 3. 조사 활동 진행

학생들이 스스로 웹사이트를 탐색하며 다음을 조사하게 할 수 있습니다:

- 감염병의 이름
- 유행한 시기
- 주요 증상
- 사람들에게 준 피해
- 예방 방법

## 💡 활용 팁

1. **개별 조사**: 학생들이 관심 있는 감염병 한 가지를 선택해 깊이 있게 조사
2. **모둠 활동**: 모둠별로 다른 감염병을 맡아 조사 후 발표
3. **비교 활동**: 여러 감염병의 공통점과 차이점 찾기
4. **독후감**: 가장 흥미로웠던 감염병에 대해 글쓰기

## 🎨 수정 방법

내용을 수정하고 싶으시면:

1. HTML 파일을 메모장이나 VS Code 등으로 열기
2. 원하는 내용 수정
3. 저장 후 웹 브라우저에서 새로고침

## 📞 문제 해결

- **페이지가 안 보여요**: 모든 파일이 같은 폴더에 있는지 확인하세요
- **사진이 안 보여요**: 인터넷 연결을 확인하세요
- **글자가 이상해요**: UTF-8 인코딩으로 저장되었는지 확인하세요
